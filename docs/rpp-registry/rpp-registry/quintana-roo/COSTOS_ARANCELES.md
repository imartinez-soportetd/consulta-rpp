# Costos y Aranceles - Registro Público Quintana Roo

> **Última actualización**: Abril 2026  
> **⚠️ Sujeto a cambios**: Verificar con oficinas antes de trámite

## 💰 Estructura de Aranceles

### Inscripción de Propiedad Inmueble

La tarifa se calcula basada en el **valor catastral** del inmueble.

| Valor Catastral | Arancel |
|-----------------|---------|
| Hasta $50,000 MXN | $300-500 MXN |
| $50,001 - $100,000 MXN | $500-800 MXN |
| $100,001 - $250,000 MXN | $800-1,200 MXN |
| $250,001 - $500,000 MXN | $1,200-1,800 MXN |
| $500,001 - $1,000,000 MXN | $1,800-2,500 MXN |
| Mayor a $1,000,000 MXN | $2,500+ (consultar) |

### Servicios Complementarios

| Servicio | Costo |
|----------|-------|
| **Anotación de Demanda** | $200 - $400 MXN |
| **Cancelación de Asiento** | $150 - $300 MXN |
| **Expedición de Certificado** | $100 - $200 MXN |
| **Certificado Adicional** | $50 - $100 MXN |
| **Búsqueda de Antecedentes** | Gratuito (en línea) - $100 MXN (certificado) |
| **Duplicado de Certificado** | $100 - $150 MXN |
| **Copia Certificada** | $150 - $250 MXN |
| **Expedición de Comprobante de No-Adeudo** | $100 - $200 MXN |

### Honorarios Notariales (Complementarios)

Los siguientes costos son **independientes** de los aranceles RPPC:

| Trámite | Rango Honorarios |
|---------|-----------------|
| **Redacción de Escritura de Propiedad** | $500 - $2,000 MXN |
| **Acta Notarial de Manifestación** | $300 - $800 MXN |
| **Poder General o Especial** | $200 - $500 MXN |
| **Acta de Disolución de Usufructo** | $400 - $1,000 MXN |
| **Copia certificada de instrumento** | $50 - $150 MXN |

## 📊 Ejemplo de Costo Total - Casos Típicos

### Caso 1: Registro de Propiedad (Casa valuada $300,000 MXN)

```
├─ Arancel RPPC: $1,500 MXN
├─ Honorarios Notariales: $1,000 MXN
├─ Certificados iniciales: $150 MXN
├─ Gestorías/trámites: $200 MXN
└─ TOTAL: ~$2,850 MXN
```

### Caso 2: Disolución de Usufructo

```
├─ Arancel RPPC (cancelación): $250 MXN
├─ Honorarios Notariales: $500 MXN
├─ Certificado nuevo: $150 MXN
└─ TOTAL: ~$900 MXN
```

### Caso 3: Anotación de Demanda

```
├─ Arancel RPPC: $300 MXN
├─ Gestión ante juzgado: $200 MXN
├─ Certificado: $100 MXN
└─ TOTAL: ~$600 MXN
```

## 📌 Formas de Pago

✅ **Aceptadas**:
- Efectivo (en oficinas)
- Transferencia bancaria (previo acuerdo)
- Cheque (en algunos casos)
- Tarjeta de débito/crédito (en algunas oficinas)

⏰ **Horarios de Pago**:
- De lunes a viernes: 9:00 AM - 2:00 PM
- Descuentos especiales: No aplican (salvo casos excepcionales)

---

## 🎓 Exenciones y Descuentos

### Posibles Exenciones
- **Inmuebles de valor muy bajo**: Consultar caso por caso
- **Trámites gubernamentales**: Verificar con dependencia
- **Bienes del patrimonio cultural**: Requiere documentación especial

### Descuentos Aplicables
- **Pago anticipado**: No aplica regularmente
- **Multitud de inscripciones**: Verificar disponibilidad

## ⚠️ Notas Importantes

❌ **No confundir**:
- Aranceles RPPC ≠ Honorarios notariales
- Ambos deben pagarse para tramite completo

🔔 **Cambios anuales**:
- Los aranceles pueden modificarse
- El Congreso Estatal publica cambios oficialmente
- Se recomienda verificar antes de cada trámite

💡 **Recomendación**:
- Solicitar cotización completa antes de proceder
- Confirmar detalles con notario autorizado
- Guardar comprobantes de pago
