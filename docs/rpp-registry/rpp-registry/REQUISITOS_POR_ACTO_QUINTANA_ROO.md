## 1️⃣ TRANSMISIONES DE DOMINIO

### 1.1 COMPRAVENTA

| Aspecto | Información |
|--------|-----------|
| **Código** | ACT-TPD-001 |
| **Descripción** | Venta de propiedad inmueble mediante contrato bilateral |
| **Base Legal** | Art. 2248 CC Quintana Roo |
| **Vigencia** | Perpetua-transferencia definitiva |
| **Plazo Promedio** | 15-25 días hábiles |
| **Costo (Derechos)** | $300-$3,200 MXN (según zona turística) |

#### DOCUMENTOS REQUERIDOS - COMPRAVENTA

**Por el Vendedor:**
- [ ] Cédula/Pasaporte vigente (válido 6+ meses si extranjero)
- [ ] RFC (Registro Federal de Contribuyentes)
- [ ] Comprobante de domicilio (no mayor 3 meses)
- [ ] Escritura de propiedad anterior (original o certificada)
- [ ] Certificado de libertad de gravámenes (RPP Q. Roo)
- [ ] Comprobante de pago predial (últimos 2 años)
- [ ] Comprobante de no adeudo municipal
- [ ] Si es persona moral: acta constitutiva + poder notario
- [ ] Firma de escribano (2 testigos si venta a extranjero)

**Por el Comprador:**
- [ ] Cédula/Pasaporte vigente
- [ ] RFC (si es persona física mexicana)
- [ ] Comprobante de domicilio (no mayor 3 meses)
- [ ] Captura de datos (fotografía actualizada)
- [ ] Si es extranjero: autorización CONAGUA (si costero)
- [ ] Si es extranjero: constancia de no restricción de propiedad
- [ ] Si es empresa: acta constitutiva + poder notario

**Documentos del Inmueble:**
- [ ] Descripción técnica completa del predio
- [ ] Plano catastral o topográfico
- [ ] Clave de Identificación del Terreno (CITA) catastral
- [ ] Avalúo catastral o INEGI
- [ ] Si hay construcción: acta de terminación de obra

**Documentación Notarial:**
- [ ] Escritura pública de compraventa notarizada
- [ ] Acta de entrega de documentos ante notario
- [ ] Comprobante de pago de honorarios notariales

**Documentación Fiscal:**
- [ ] Comprobante de pago ISR (10% comprador)
- [ ] Comprobante de pago traslado de dominio estatal
- [ ] RIF (Registro de Información Fiscal) actualizado

#### ESPECIFICIDADES QUINTANA ROO - COMPRAVENTA

**ZONAS TURÍSTICAS (Cancún, Playa del Carmen, Cozumel, Tulum, Isla Mujeres):**
- Derechos incrementados 30-35%
- Requiere documento adicional de zona turística
- Verificación de no restricción federal
- Posible pago ambiental: $500-$1,500

**ZONAS COSTERAS:**
- Autorización CONAGUA obligatoria (15 días aprox.)
- Permisos PROFEPA si hay patrimonio natural
- Depósito ambiental o póliza: $500-$2,000
- Certificado de no afectación a ecosistema
- Comprobante de resolución ambiental

**BIENES EJIDALES:**
- Certificado PROCEDE de inafectabilidad
- Acta de asamblea ejidal aprobando venta
- Solicitud ante SRA (Secretaría de Reforma Agraria)
- Documento de ocupación histórica
- Registro en padrón ejidal

**PROPIEDAD EXTRANJERA:**
- RFC extranjero (trámite ante SAT)
- Permiso SHCP (Secretaría Hacienda)
- Limitación: No permitida a menos 50 km de costa, 100 km frontera
- Fideicomiso como alternativa legal
- Comprobante de nacionalidad verificada

- **Donatarios extranjeros**: Mismas restricciones que compraventa
- **Cargas/Condiciones**: Permisibles, requieren aclaración en escritura

- [ ] Comprobantes de no adeudo municipal

**Del Predio A:**
- [ ] Descripción técnica completa
- [ ] Plano catastral (CITA)
- [ ] Avalúo fiscal/INEGI
- [ ] Especificación de mejoras/construcciones

**Del Predio B:**
- [ ] Descripción técnica completa
- [ ] Plano catastral (CITA)
- [ ] Avalúo fiscal/INEGI
- [ ] Especificación de mejoras/construcciones

**Documentación Notarial:**
- [ ] Escritura única de permuta (ambos actos)
- [ ] Acta de consentimiento mutuo
- [ ] Comprobante de entrega de posesión
- [ ] Si hay compensación monetaria: especificar cantidad

**Impuestos (según monto mayor):**
- [ ] Comprobante de ISR (si aplica)
- [ ] Traslado de dominio estatal
- [ ] Actualización catastral para ambos

#### ESPECIFICIDADES QUINTANA ROO - PERMUTA

- Se considera como dos compraventas simultáneas
- Aplicar derechos más altos si uno es en zona turística/costera
- Ambos inmuebles deben estar libres de controversia
- Si hay extranjero: aplicar régimen de inversión extranjera

| **Base Legal** | Art. 1340-1602 CC Quintana Roo |
| **Vigencia** | Perpetua-asignación definitiva |
| **Plazo Promedio** | 30-45 días hábiles (incluye sucesión) |
| **Costo (Derechos)** | $200-$1,200 MXN (muy reducido) |

#### DOCUMENTOS REQUERIDOS - ADJUDICACIÓN HEREDITARIA

**Por los Herederos:**
- [ ] Cédulas vigentes de todos los herederos
- [ ] RFC de todos (si aplica)
- [ ] Actas de nacimiento (comprobación de calidad de heredero)
- [ ] Acta de matrimonio del causante (si hay viuda/viudo)
- [ ] Actas de matrimonio de herederos (si son casados)

**Del Causante (Descente):**
- [ ] Acta de defunción oficial (original del registro civil)
- [ ] Última voluntad (testamento si existe)
- [ ] Si no hay testamento: solicitud de sucesión intestada

**Del Inmueble:**
- [ ] Escritura anterior de propiedad del causante
- [ ] Certificado de libertad de gravámenes (nombre causante)
- [ ] Descripción técnica del inmueble
- [ ] Plano catastral (CITA)
- [ ] Comprobante de pago último predial

**Documentación Judicial/Notarial:**
- [ ] Sentencia de sucesión (del Juzgado Familiar)
- [ ] Acta de proposición hereditaria
- [ ] Acta notarial de adjudicación específica
- [ ] Comprobante de depósito de autos (si hubo)

**Impuestos/Fiscal:**
- [ ] Dictamen fiscal (si herencia sujeta a impuesto)
- [ ] Comprobante de pago de derechos sucesorios
- [ ] RFC de herederos (si corresponde)

#### ESPECIFICIDADES QUINTANA ROO - ADJUDICACIÓN

- Derecho de preferencia: coherederos pueden comprar parte de otros
- Si bienes en zonas turísticas: verificar que herederos cumplen requisitos
- Si heredero es extranjero: aplicar restricciones de propiedad foreigners
- Posible impuesto estatal sobre herencia (varía por municipio)

- Documentación de primer gravamen vigente

### 2.2 SERVIDUMBRE (Paso, Aguas, Luz)

| Aspecto | Información |
|--------|-----------|
| **Código** | ACT-GDR-002 |
| **Descripción** | Derecho sobre predio ajeno  (sirviente) para utilidad de otro (dominante) |
| **Base Legal** | Art. 1084-1155 CC Quintana Roo |
| **Vigencia** | Perpetua o término convencional |
| **Plazo Promedio** | 15-25 días hábiles |
| **Costo (Derechos)** | $300-$800 MXN (simple), $400-$1,200 (compleja) |

#### DOCUMENTOS REQUERIDOS - SERVIDUMBRE

**Partes (Prediante y Prediario):**
- [ ] Cédulas vigentes de ambos
- [ ] RFC de ambos
- [ ] Comprobantes de domicilio actual
- [ ] Acta de consentimiento del dueño sirviente
- [ ] Poderes notariales (si están representados)

**Predio Dominante (que se beneficia):**
- [ ] Escritura de propiedad
- [ ] Descripción técnica
- [ ] Plano catastral (CITA)
- [ ] Comprobante de dominio vigente

**Predio Sirviente (limitado):**
- [ ] Escritura de propiedad
- [ ] Descripción técnica
- [ ] Plano catastral (CITA)
- [ ] Especificación del área afectada

**Documentación Técnica (CRÍTICO):**
- [ ] Plano topográfico detallado con ambos predios
- [ ] Levantamiento técnico mostrando paso/área servil
- [ ] Especificación de ancho, dirección, distancias
- [ ] Especificación de tipo de servidumbre (paso, agua, luz, etc.)
- [ ] Descripción física de instalaciones (tuberías, canales, etc.)

**Documentación Notarial:**
- [ ] Escritura de constitución de servidumbre
- [ ] Acta de acuerdo entre propietarios
- [ ] Descripción técnica integrada en acta
- [ ] Documento de limitaciones/restricciones (si aplica)

**Impuestos/Documentación:**
- [ ] Comprobante de pago de derechos
- [ ] Actualización catastral con servidumbre
- [ ] RFC de ambas partes

#### ESPECIFICIDADES QUINTANA ROO - SERVIDUMBRE

**TIPOS COMUNES EN Q. ROO:**
1. **Paso**: Acceso a predio enclaustrado
   - Ancho mínimo: 3-5 metros
   - Documentación de necesidad: acta administrativa
   
2. **Aguas**: Conducción/desagüe en zona costera
   - Comprobante CONAGUA autorización de obra
   - Especificación de caudal (si aplica)
   - Cédula de derechos de agua (si existe)
   
3. **Acueducto/Alcantarilla**: Tuberías subterráneas
   - Plano de obras subterráneas
   - Especificación de profundidad y materiales
   - Documentación de mantenimiento
   
4. **Luz/Aire**: Fenestras o miradores
   - Acta de no obstaculización
   - Especificación de ángulos visuales
   - Documento de vigencia temporal/perpetua

- Servidumbre continua vs discontinua: afecta registro (continua más fácil adquisición por usucapión)
- Servidumbre aparente vs no aparente: presunción legal diferente
- Plazo extendido si requiere permisos ambientales adicionales
