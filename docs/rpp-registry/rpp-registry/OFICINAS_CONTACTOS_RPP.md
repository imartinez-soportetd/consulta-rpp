```
✅ CONSULTAS:
  • Búsqueda de propietario
  • Búsqueda por clave catastral (CITA)
  • Historial de propiedades
  • Estado de trámites

✅ SOLICITUDES:
  • Certificado de propiedad
  • Certificado de gravámenes
  • Certificado tradición
  • Copia de folio

✅ SEGUIMIENTO:
  • Estado de solicitudes
  • Descarga de documentos
  • Notificaciones
```

**Pasos para Usar SIQROO:**
1. Acceder a www.sispol.quintanaroo.gob.mx
2. Crear cuenta (nombre, email, teléfono)
3. Seleccionar tipo de servicio
4. Ingresar datos del inmueble
5. Realizar pago (si aplica)
6. Descargar/ Recibir documento

**Costo de Servicios en Línea:**
- Consultas: Gratuito
- Certificados solicitados: Mismo arancel (-20% por digital)

## ⚠️ NOTAS IMPORTANTES

### Antes de Visitar Oficina:

1. **Verificar Cambios de Dirección/Horarios**
   - Llamar a central antes de viajar
   - Confirmar que no hay festivos especiales
   - Preguntar por citas agendadas

2. **Documentación Completa**
   - Verificar lista de requisitos específica
   - Incluir copias adicionales
   - Revisar que firmas estén autenticadas

3. **Formas de Pago**
   - Llevar efectivo preferentemente
   - Verificar si aceptan tarjeta/transferencia
   - Solicitar recibos oficiales

4. **Para Extranjeros**
   - Pasaporte vigente
   - RFC México (si aplica)
   - Autorización específica (si costero)
