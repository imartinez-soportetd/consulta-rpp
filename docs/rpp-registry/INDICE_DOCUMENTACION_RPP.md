## 🎓 Guías de Uso por Usuario

### Para Abogados Especialistas
1. Consultar `DICCIONARIO_ACTOS_REGISTRABLES.md` para bases legales
2. Verificar derechos en `DERECHOS_COSTOS_*.md` correspondiente
3. Usar `GUIA_COMPARATIVA_PUEBLA_QROO.md` para asesoría jurisdiccional

### Para Gestores de Trámites
1. Identificar acto en `DICCIONARIO_ACTOS_REGISTRABLES.md`
2. **Si es Q. Roo**: Consultar `REQUISITOS_POR_ACTO_QUINTANA_ROO.md` para lista completa
3. Confirmar ubicación del inmueble (Puebla o Q. Roo)
4. **Consultar `OFICINAS_CONTACTOS_RPP.md`** para contactar la oficina correcta
5. Calcular costos exactos en tarifa de derechos
6. Usar checklist de requisitos por acto
7. Proceder con inscripción ante RPP

### Para Notarios Públicos
1. Revisar definición de acto en diccionario
2. **Si es Q. Roo**: Consultar `REQUISITOS_POR_ACTO_QUINTANA_ROO.md` para requisitos completos
3. Consultar derechos aplicables en tarifa estatal
4. **Usar `OFICINAS_CONTACTOS_RPP.md`** para teléfono/correo de coordinación
5. Solicitar al cliente pago de derechos
6. Preparar escritura notarial según formato registral

### Para Asesores Inmobiliarios
1. Determinar si es Puebla o Q. Roo
2. **Usar `OFICINAS_CONTACTOS_RPP.md`** para ubicación de oficina relevante
3. Usar comparativa para presupuestar al cliente
4. Consultar casos similares en análisis de costos
5. Recomendar estrategia de optimización
6. Presentar propuesta integral con contactos

### Para Inversores/Compradores
1. Leer `GUIA_COMPARATIVA_PUEBLA_QROO.md` completa
2. **Consultar `OFICINAS_CONTACTOS_RPP.md`** para contactar directamente con preguntas
3. Usar checklist decisional de jurisdicción
4. Calcular costos totales con ejemplos
5. Entender tiempo y proceso esperado
6. Preparar presupuesto con contingencias (+15-20%)

## 🔍 Búsqueda Rápida por Concepto

### Por Tipo de Inmueble
- **Residencial urbano** → Ver Compraventa, Hipoteca
- **Comercial** → Ver Compraventa, Servidumbre
- **Ejidal** → Ver procedimientos especiales Q. Roo
- **Costero** → Ver regulaciones especiales Q. Roo
- **Herencia/Sucesión** → Ver Adjudicación

### Por Tipo de Operación
- **Compra-venta** → Compraventa + posible Hipoteca
- **Financiamiento** → Hipoteca + Anotación Preventiva
- **Transferencia gratuita** → Donación
- **Intercambio** → Permuta
- **Resolución conflicto** → Anotación Demanda → Sentencia

### Por Urgencia
- **Urgente (1-5 días)** → Embargo, Cancelación Embargo
- **Normal (10-15 días)** → Hipoteca, Cancelación
- **Largo (20-45 días)** → Compraventa, Sucesión

## 💡 Casos de Uso Comunes

### Caso 1: Primera vez comprando casa en Puebla
```
Acciones recomendadas:
1. Revisar sección "Compraventa" en diccionario
2. Confirmar si requiere hipoteca
3. Consultar "DERECHOS_COSTOS_PUEBLA.md"
4. Calcular presupuesto total (~$16-23K adicional)
5. Coordinar con Notario y banco
```

### Caso 2: Inversión turística en Cancún
```
Acciones recomendadas:
1. Leer "GUIA_COMPARATIVA_PUEBLA_QROO.md" sección Q. Roo
2. Revisar regulaciones costeras y extranjeras
3. Consultar "DERECHOS_COSTOS_QUINTANA_ROO.md"
4. Calcular con +$500-1,500 ambiental
5. Considerar fideicomiso
```

### Caso 3: Herencia múltiple en ambos estados
```
Acciones recomendadas:
1. Obtener sentencia de sucesión
2. Listar todos los inmuebles (Puebla + Q. Roo)
3. Consultar derechos por estado
4. Agrupar adjudicaciones
5. Solicitar descuentos por volumen
```

### Caso 4: Cancelación de hipoteca prestada
```
Acciones recomendadas:
1. Confirmar pago total de deuda
2. Solicitar carta de liberación al banco
3. Consultar derechos de cancelación (~$100-300)
4. Presentar ante RPP
5. Plazo: 5-10 días normalmente
```
